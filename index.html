    How it works

    Simple setup, visible behavior.

      Setup Guide

      Match the visible signal with RapidyCore.

      Configure the in-game highlight color and match it with RapidyCore's external color trigger.

      The game shows a visible color signal, RapidyCore detects that color externally from the screen, and then uses the configured normal Windows input action.

      Trigger Bot reacts to the configured color in the scanned screen area. Any object or effect with the same or similar color, such as swords, visual effects, UI highlights, or particles, may also be detected.

      RapidyCore uses external screen-based color detection. It does not inject into Roblox, read memory, or execute code inside the game.

      Safety and transparency

      Local by design, clear about what it does.

      RapidyCore is designed around external screen-based detection and normal Windows input. It does not require an account, cloud backend, or hidden background services.
